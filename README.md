# RBT Interview 面试题库系统设计方案

RBT Interview 面试题库系统旨在为红黑树学生乃至所有中国留学生提供面试准备的在线资源。系统分为管理端和用户端两大部分，实现题目管理、用户管理、学习追踪等核心功能。

本系统目标用户：

 - 红黑树教师：用来上传题目，以及编写答案
 - 红黑树学生：注册登录，参考历年题目以及相关面试题
 - 所有留学生：TODO，未来可以做成红黑树学生免费一年，其他留学生付费注册会员

## 系统架构

### 整体架构
采用前后端分离的架构模式：

```txt
前端：React + TypeScript
后端：Node.js (Express)
数据库：PostgreSQL
```

### 系统结构

```txt
├── 管理端 (Admin Portal)
│   ├── 用户管理
│   ├── 题目管理
│   ├── 标签/分类管理
│   └── 数据统计/分析
│
├── 用户端 (User Portal)
│   ├── 题目浏览/搜索
│   ├── 个人收藏
│   ├── 评论互动
│   └── 个人中心
│
├── 后端 API 服务 (Express)
│   ├── 认证授权模块
│   ├── 用户模块
│   ├── 题目模块
│   ├── 评论模块
│   └── 收藏模块
│
└── 数据库 (PostgreSQL)
    ├── 用户表
    ├── 题目表
    ├── 评论表
    └── 收藏表
```

## 功能需求详细说明

### 管理端功能

#### 1. 用户管理

- 用户列表：查看所有注册用户，支持分页、搜索和筛选

- 用户详情：查看单个用户信息，包括基本资料、活跃度等

- 用户权限：设置用户角色（普通用户/管理员），冻结/解冻账号

- 数据导出：导出用户数据（CSV/Excel格式）

#### 2. 题目管理

- 题目列表：查看所有题目，支持分页、搜索和多维度筛选

- 题目创建/编辑：

- 题目标题和描述（支持富文本编辑）

- 题目难度设置（简单/中等/困难）

- 标签和分类关联

- 代码示例（多语言支持）

- 答案和解析（支持富文本）

- 题目导入/导出：批量导入或导出题目

- 题目上下线：控制题目是否可见

#### 3. 标签/分类管理

- 标签管理：创建、编辑、删除标签

- 分类管理：创建分类树，支持多级分类

- 关联分析：查看各标签/分类下的题目数量和热度

#### 4. 数据统计/分析

- 使用统计：用户增长趋势、活跃度分析

- 内容统计：热门题目排行、评论热度分析

- 操作日志：记录管理员操作，支持审计

### 用户端功能

#### 1. 用户认证

- 注册：邮箱/手机号注册

- 登录：账号密码登录，第三方登录（可选：GitHub、Google）

- 找回密码：通过邮箱验证找回

#### 2. 题目浏览/搜索

- 首页推荐：根据热度、最新上线等维度推荐题目

- 分类浏览：按技术领域、难度等分类浏览

- 高级搜索：支持标题、内容、标签多维度搜索

- 题目详情：查看完整题目内容、解答和讨论

#### 3. 个人收藏

- 收藏管理：添加/取消收藏，创建收藏夹

- 收藏列表：查看已收藏题目，支持自定义排序

- 收藏分享：分享收藏夹（可选功能）

#### 4. 评论互动

- 发表评论：对题目发表评论/解答

- 回复互动：回复其他用户评论

- 点赞/投票：对评论进行投票

- 举报功能：举报不当内容

#### 5. 个人中心

- 个人资料：查看/编辑个人信息

- 活动记录：浏览历史、评论记录

- 学习进度：已完成题目统计、难度分布

#### 6. 奖励机制

- 完成题目获得积分。

- 积分兑换奖励，激励用户参与。

- 排行榜功能，展示用户排名。

### 技术选型

#### 前端技术栈

- 框架：React 18

- UI 组件库：Ant Design

- 样式解决方案：Styled Components

- 路由：React Router v7

- HTTP 请求：Axios

- 表单处理：Formik / React Hook Form

- 代码编辑器：Monaco Editor

- 构建工具：Vite

#### 后端技术栈

- 框架：Express.js

- API 文档：Swagger / OpenAPI

- 认证授权：JWT + Cookie

- 数据校验：express-validator / Joi

- 日志：Winston / Morgan

- 文件上传：Multer + AWS S3/阿里云OSS

- 安全：helmet, cors, rate-limiting
